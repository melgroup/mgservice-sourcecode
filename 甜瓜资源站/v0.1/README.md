# 甜瓜资源站 部署教程

一个简易的甜瓜ZIP资源网站，支持上传、下载和管理资源

## 环境要求

- PHP 7.4 或更高版本
- MySQL 5.7 或更高版本 (或MariaDB 10.2+)
- 支持PDO MySQL扩展的Web服务器 (Apache/Nginx)

## 部署步骤

### 1. 上传文件

将以下文件上传到服务器网站目录：

- `install.php`
- `index.php`
- `admin.php`

### 2. 创建数据库

在MySQL中创建一个空数据库，例如命名为 `melon`

记下数据库主机、用户名、密码和数据库名

### 3. 运行安装向导

在浏览器访问 `http://网站域名/install.php`

填写表单：

- 数据库主机
- 数据库用户名
- 数据库密码
- 数据库名称
- 管理员密码 (用于登录管理后台)

点击「开始安装」，程序将自动创建数据表并生成配置文件 `config.php`

### 4. 删除安装文件

安装完成后，**务必删除 `install.php`**，以防被他人恶意利用

### 5. 完成

访问 `http://网站域名/index.php` 即可正常使用

管理后台地址为 `http://网站域名/admin.php`，使用安装时设置的管理员密码登录

## 目录结构

```

/网站目录
├── uploads/         (上传文件存储目录，自动创建)
├── config.php       (配置文件，安装后自动生成)
├── index.php        (主页面)
├── admin.php        (管理员后台)
└── install.php      (安装向导，部署后请删除)

```

## 常见问题

**Q: 安装时报数据库连接错误**  
A: 检查数据库主机、用户名、密码和数据库名是否正确，确认数据库已创建

**Q: 上传失败**  
A: 确保 `uploads` 目录有写入权限（权限设为 755 或 775）

**Q: 想重新安装**  
A: 删除 `config.php` 文件，重新访问 `install.php`
